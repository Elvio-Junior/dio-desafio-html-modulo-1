O que são as Tags?

As Tags são usadas para informar ao navegador a estrutura do site. São interpretadas pelo navegador para compor a estrutura e o visual da página

Sempre estão dentro dos sinais de < >

Podem ter ou não fechamento. Caso tenha fechamento a denotação é < tag > para abertura e < /tag > para o fechamento. Sem fechamento a denotação é < tag />

Outro ponto importante é que a Tag pode conter Atribuito. O Atributo é usado para modificar a estrutura ou funcionalidade da Tag e podem ser globais ou específicos

Tipos de Tags

Podemos ter os seguintes tipos de Tags

Tags de Texto

Definem textos, estilos de fontes, paragrafos entre outros

  • <p></p>: compõe um paragrafo
  • <hr/>: compões uma linha horizontal
  • <i></i>: destaca uma parte do texto em Italico. Exemplo: Italico
  • <u></u>: destaca uma parte do texto em Sublinhado. Exemplo: Sublinhado
  • <strong></strong>: destaca uma parte do texto em Negrito. Exemplo: Negrito
  • <mark></mark>: destaca uma parte do texto colorindo o mesmo. Exemplo: Colorindo
  • <small></small>: destaca um texto como comentário. Exemplo:

    HTML HyperText Markup Language

  • <sub></sub>: destaca um texto como subscrito. Exemplo: H20
  • <sup></sup>: destaca um texto como sobrescrito. Exemplo: 22
  • <blockquote></blockquote>: destaca um texto como uma citação. Exemplo:
    HTML-HyperText Markup Language
  • <font></font>: Esta tag não é mais recomendada, necessitando de verificar a compatibilidade com os navegadores. Criado para definir o tamanho da fonte. Recommendado utiliza propriedade do CSS para estilizar o texto.
  • <del></del>: destaca um texto com um risco. Exemplo: Riscado
  • <abbr></abbr>: destaca um texto com uma abreviação. Exemplo: Abreviação
Indice

Tags de Lista

Definem as listas a serem utilizadas

  • <ol></ol>: compõe um lista ordenada. Exemplo:
    1. Item 1
    2. Item 2
  • <ul></ul>: compõe um lista não ordenada. Exemplo:
    • Item 1
    • Item 2
  • <li></li>: compõe um ou mais itens de uma lista (ordenada ou não). Exemplo:
  • Item 1
  • Item 2

Tags de Link

Responsável por fazer uma ligação a um documento da mesma pagina e/ou pagina de outro dominio

  • <a></a>: para realizar uma chamada de um link ou pagina do mesmo projeto
